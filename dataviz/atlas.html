Key rule: Always check if log-transform is needed. Power-law distributions (city populations, word frequencies, wealth) look meaningless on a linear scale.

Note: Scatter plots with >10k points need density encoding (hexbin, contour, alpha blending) to avoid overplotting.

Avoid pie charts with >5 slices or 3D pie charts (always). Treemaps are almost universally better.

05

Temporal

Data with a time dimension.

Line Chart

Spiral Plot

Multi-Line Chart

Horizon Chart

Seasonal Decomposition

Radial Clock Plot

★

Creative coding opportunity: Time-series as particle trails, physics simulations, or sound. Map temporal data to animation frames in p5.js or D3 transitions.

06

Spatial / Geographic

Data with coordinates, place names, or spatial relationships.

Dot Map

Density Heatmap

Choropleth

Cartogram

Flow Map

Arc Map

★

Tools: deck.gl for WebGL performance with large point datasets, kepler.gl for quick exploration, D3 + topojson for bespoke cartography.

07

Network / Graph

Connections, relationships, flows between entities.

Force-Directed Graph

Arc Diagram

Adjacency Matrix

Bundled Edge Layout

Tree Layout

Radial Tree

Sankey Diagram

Alluvial Diagram

⚠

Warning: Force-directed layouts are beautiful but can mislead — node proximity doesn't necessarily mean similarity. Always clarify what the layout algorithm optimizes for.

Embedding projections (t-SNE, UMAP) create stunning 2D landscapes from high-dimensional data. Clusters emerge naturally — excellent for data art.
